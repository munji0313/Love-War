프로젝트 주제 : 3인칭 슈팅 게임(사랑과 전쟁)

1-2. 개발 배경 및 필요성
- C언어 및 알레그로 라이브러리를 활용한 게임 프로젝트를 통해 팀 협업 능력 및 C언어 개발 능력 함양

1-3. 게임 시나리오
- 닉네임 입력 후 start 버튼 클릭하여 게임 시작
- 방향키 입력으로 캐릭터 이동 및 z키를 통한 원거리 공격
- 총 4단계의 stage로 구성
- 적은 2가지 종류로 구성
- 최종 보스 클리어 이후 점수 공개

💡2. 팀원 소개
- 이재혁 : UI 제작, 맵 구현
- 강민지 : 무기 시스템, 목숨 및 부활 시스템 구현
- 변성호 : 적 시스템 구현
- 함효정 : 메인 캐릭터 조작, 점수 구현
